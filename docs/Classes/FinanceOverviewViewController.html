FinanceOverviewViewController

public final class FinanceOverviewViewController : UIViewController
extension FinanceOverviewViewController: UIScrollViewDelegate

A view controller for displaying an overview of a user’s finances.

A FinanceOverviewViewController displays an interface with sections of cards and graphs representing the user’s latest data.

Note

It is suitable for use in a UINavigationController.
